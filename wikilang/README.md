Program **wikilang** displays link to the alternative language version.
	Usage:

	go2 lang url 

	Example: 

	go2 cs "https://en.wikipedia.org/wiki/Cassiopeia_(constellation)" 

	go2 cs https://en.wikipedia.org/wiki/Tau_function 
.
