Program **randomgroups** randomly divides people into a number of groups of the maximum size.

	Usage:

	-h this help

	-d demo

	-f filename         #json

	-g group-max-size   #default is 2 

	Examples:

	-d 

	-g 3 -d 

	-g 4 -f people.json
